## 🎯 The Challenge

**"Build an application where two or more AI agents autonomously discover each other, negotiate, and exchange value through BSV micro-payments — solving a real-world problem."**

### 📝 Requirements

- ✅ **At least 2 AI agents** with their own BSV wallets
- ✅ **At least 1.5M transactions** on a 24-hour designated window for eligibility
  - Transactions must be meaningful to the application's functionality
  - ⚠️ Artificial inflation will result in disqualification
- ✅ **Agents discover each other** using BRC-100 wallets and identity
- ✅ **Agents transact autonomously** (MessageBox P2P or direct payments)
- ✅ **Solves a real, identifiable problem**
- ✅ **Human-facing web UI** showing agent activity

### 🔥 Scale Requirement

For eligibility, agents must demonstrate at least **1.5 million transactions recorded on-chain** within a designated 24-hour window.

## 🚀 Example Project Ideas

### 📊 Data Marketplace
- Agents buy and sell datasets
- Pay per query or per record
- Value exchange through micro-payments

### 🤖 AI Service Broker
- Router agent pays specialist agents per task
- Translation, summarization, code review services
- Tokenized service marketplace

### ⚡ Compute Settlement
- Distributed compute nodes settle micro-payments
- GPU/CPU time payments
- Resource allocation optimization

### 💬 Messaging & Communication
- P2P messaging with tokenized features
- Premium message types via micro-payments
- Subscription-based agent capabilities

### 📈 Prediction Markets
- Sports, events, and outcomes predictions
- Fractional ownership of predictions
- Peer-to-peer betting marketplace

## 🛠️ Tech Stack

Everything you need to build agentic payment systems on BSV:

- **BSV Blockchain** - Native Bitcoin Ordinals and scripting
- **BRC-100 Wallets** - Identity and discovery
- **Agent Communication** - P2P messaging and coordination
- **Micro-payments** - Native BSV token transfers
- **Human UI** - Web interface for monitoring
